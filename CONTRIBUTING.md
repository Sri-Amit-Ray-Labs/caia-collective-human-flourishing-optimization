# Contributing to CHFOA - Collective Human Flourishing Optimized AI Algorithm

Thank you for your interest in contributing to the **Collective Human Flourishing Optimized Algorithm (CHFOA)** project.

This project is inspired by **Sri Amit Ray's Compassionate AI Architecture (CAIA)** and **Human Flourishing Optimization Algorithm (HFOA)**, and contributions should reflect these principles.

## Core Values

Please align your contributions with:

- **Ahimsa in Code:** Avoid harmful or exploitative behavior in algorithms and data.
- **Equity and Inclusion:** Protect vulnerable groups, reduce bias, and promote fairness.
- **Transparency and Accountability:** Provide clear explanations and auditable logic.
- **Long-Horizon Flourishing:** Consider long-term wellbeing, not just immediate metrics.

## Contribution Workflow

1. **Discuss**
   Open an issue describing the feature, bug fix, or experiment you want to contribute.

2. **Design**
   Propose design changes in terms of CHFOA modules:
   - FSM, CEE, ECL, CDI, ACLL, Transformer model, benchmarks, or data.

3. **Implement**
   - Follow code style of existing modules.
   - Write tests in `tests/`.
   - Document behavior in docstrings and comments.

4. **Review**
   Submit a pull request:
   - Link to relevant issues.
   - Explain how your changes align with Sri Amit Ray CAIA/HFOA principles.
   - Include any new references used.

## Types of Contributions

- New CHFOA modules (e.g., new flourishing domains, compassion heads).
- Improved synthetic data generators or benchmarks.
- Enhanced transformer models for flourishing-based decision-making.
- Web UI enhancements for CDI (human-in-the-loop review).
- Documentation and tutorials.

## Ethical Checklist

Before submitting:

- Have you considered the impact on vulnerable groups?
- Have you documented any fairness, harm, or drift assumptions?
- Are any new datasets compliant with privacy and consent requirements?
- Are your changes auditable and explainable?

Thank you for helping build compassionate, flourishing-centered AI systems in the spirit of Sri Amit Ray's vision.
